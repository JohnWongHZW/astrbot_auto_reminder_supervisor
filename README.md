# 自动提醒监督助手

> **本插件大部分由 AI 辅助开发，有问题轻喷尽量修。**

> **作者：流浪本子商人**

## 基于时间调度和消息监听的群聊监督插件，支持定时提醒和持续监督两种模式。

## 功能特性

- **定时提醒模式**: 在指定时间主动@用户发送提醒，支持一次性和每天重复
- **持续监督模式**: 在指定时间段监听用户发言，发一次消息就提醒一次（无冷却时间）
- **支持跨天时间区间**: 例如 `22:00-06:00` 表示晚上10点到次日早上6点
- **多用户独立配置**: 每个用户可配置多条独立规则
- **可视化配置面板**: 通过 WebUI 图形化界面管理规则

## 安装方法

**推荐直接在插件市场安装**

1. 将插件文件夹 `astrbot_auto_reminder_supervisor` 复制到 AstrBot 的 `data/plugins/` 目录下
2. 在 WebUI 的插件管理页面找到该插件
3. 点击重载插件或重启 AstrBot

## 命令使用

### 查看帮助

```
/rs 帮助
```

### 添加定时提醒

在指定时间主动提醒用户，支持一次性或每天重复。

```
/rs 添加提醒 @用户 时间 HH:MM 内容 [一次性|每天]
```

**示例:**
```
/rs 添加提醒 @大黑塔 22:00 该睡觉了，黑塔女士 每天
/rs 添加提醒 @大黑塔 13:20 有场会议，黑塔女士 一次性
```

> **重复类型说明**: `一次性` 表示只触发一次后自动禁用，`每天` 表示每天定时触发。默认每天。

### 添加持续监督

在指定时间段内监听目标用户发言，用户每发一条消息就自动@并质问。

```
/rs 添加监督 @用户 开始时间 HH:MM 结束时间 HH:MM 触发回复内容
```

**示例:**
```
/rs 添加监督 @大黑塔 22:00 06:00 黑塔女士，头发掉光！黑塔女士，聪明秃顶！
```

> **跨天说明**: 当结束时间早于开始时间时，系统会自动识别为跨天场景。例如 `22:00-06:00` 表示晚上22:00到次日早上06:00。

### 查看规则列表

```
/rs 列表
```

### 启用/禁用/删除规则

支持多选操作，同时管理多条规则：

```
/rs 启用 1 2 3      # 同时启用规则1、2、3
/rs 禁用 1 2 3      # 同时禁用规则1、2、3
/rs 删除 1 2 3      # 同时删除规则1、2、3
```

## WebUI 配置面板

在插件管理页面可以通过可视化表格管理规则：

### 定时提醒
| 字段 | 说明 |
|------|------|
| 用户QQ号 | 被提醒用户的QQ号（必填） |
| 用户昵称 | 用户昵称（选填） |
| 群号 | 发送目标群号（选填） |
| 是否启用 | 规则是否启用 |
| 重复类型 | 一次性/每天 |
| 提醒时间 | HH:MM 格式 |
| 提醒内容 | 发送给用户的提醒文本 |

### 持续监督
| 字段 | 说明 |
|------|------|
| 用户QQ号 | 被监督用户的QQ号（必填） |
| 用户昵称 | 用户昵称（选填） |
| 群号 | 发送目标群号（选填） |
| 是否启用 | 规则是否启用 |
| 重复类型 | 一次性/每天 |
| 开始时间 | HH:MM 格式 |
| 结束时间 | HH:MM 格式 |
| 回复内容 | 用户发消息后回复的内容 |

## 注意事项

1. 使用@用户时请确保用户在群内
2. 建议合理设置监督规则，避免对用户造成骚扰
3. 监督规则仅在对应群聊内生效
4. 插件卸载后规则数据不会自动删除
